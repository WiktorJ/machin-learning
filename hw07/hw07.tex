\section{Problem 1}
It is \textbf{not} guaranteed that all samples will have correct labels assigned. In Soft-margin SVM the objective
function provides trade-off between margin size and classification accuracy. 
So it might happen that for linearly separable dataset with some parameter $C$ 
misclassification will yield a better (smaller) value of objective function.

\subsection{Example}
Let's consider following one dimensional training dataset: 
\begin{center}
    \begin{tabular}{ |c|c| } 
     \hline
     X & Y  \\ 
     \hline
    -10 & -1 \\ 
    \hline
    0 & -1 \\ 
    \hline
    1 & 1 \\
     \hline
    \end{tabular}
\end{center}
Objective function for soft-margin SVM:
\begin{equation}
    f_0(w,b,\xi) = \frac{1}{2} w^Tw + C\sum_{i=1}^N{\xi_i}
\end{equation}

To split correctly the hyperplane $P$ should be equal to $0.5$ 
and size of the margin $m=1$. 
From equation $m=\frac{2}{||w||}$ we can calculate that $||w||=2=w$.
We also know that all $\xi_i$ are equal to $0$ since all points are correctly
classified. Thus:
\begin{equation*}
    f_0(w,b,\xi) = \frac{1}{2} 2*2 = 2
\end{equation*}
Let's see now what happens when the decision boundary is moved to $P=-4.5$
taking $-10$ and $1$ as support vectors and misclassifying $x=0$\\
In this case $m=11$ and $w=\frac{2}{11}$. Also $\xi_1$ and $\xi_3$ are equal to
$0$ (as those points are classified correctly). $\xi_2$ is misclassified and
for optimal solution we have 
\begin{equation}
    \xi_2=1-y_2(wx_2 + b) = 1+b
\end{equation}
We can also calculate $b$ with $d=-\frac{b}{||w||}$ as we know distance from
origin.
\begin{equation}
    \begin{gathered}
    -4.5 = - \frac{b}{\frac{2}{11}} \\
    -b = \frac{9}{2} * \frac{2}{11} \\
    b = \frac{9}{11}
\end{gathered}
\end{equation} 
Plugging it all back to the objective function:
\begin{equation}
    f_0(w,b,\xi) = \frac{1}{2} * (\frac{2}{11})^2 + C * \frac{20}{11} = 
    \frac{2}{121} + C * \frac{20}{11}
\end{equation}
Thus for all $C < \frac{12}{11}$ Soft-margin SVM will misclassify $x=0$
regardless of the fact that the dataset is linearly separable.

\section{Problem 2}
For soft-margin SVM we have:
\begin{equation}
    f_0(w,b,\xi) = \frac{1}{2} w^Tw + C\sum_{i=1}^N{\xi_i}
\end{equation}
with condition 
\begin{equation}
    \xi_i \geq 0
\end{equation}
Thus for $C<0$, $C\sum_{i=1}^N{\xi_i} < 0$.
As this is minimization problem our solution will favour big values of $\xi_i$.
The consequence of this is the bigger misclassification the "better" the result.
Which contradicts the whole idea of using SVM as classifying method.

Algorithm compares every possible pair of characters from strings $x$ and $y$

\subsection{Kernel}
Let's consider a function:
\[ f(x,y) =  \begin{cases} 
    1 & x = y \\
    0 & x \neq y 
 \end{cases}
\]
Also lets transform the alphabet in a way that every character is assigned
an unique natural number 
(i.e. first character is encoded as $1$, second as $2$ ... and last character is encoded by $v$). 
In this way we will work on vector of numbers rather
then vector of characters. \\\\
Now we can write down Algorithm 1 in more compact form:
\begin{equation}
    k(x,y) = \sum_{i=1}^m\sum_{j=1}^nf(x_i, y_j)
\end{equation}
As sum of valid kernels is also a valid kernel in order to show
validity of $k$ it's enough to prove that $f$ is valid.\\\\
To do this let's consider a mapping:
\begin{equation}
    \phi: \mathbb{R} -> \mathbb{R}^v
\end{equation}
\begin{equation}
    \phi(x) = [max{(0, (-|x-1| + 1))}, max{(0, (-|x-2| + 1))},..., max{(0, (-|x-v| + 1))}]
\end{equation}
So for $x \in \mathbb{N}$, $\phi(x)$ is a $v$ dimensional vector with all but one elements 
equal $0$. The only non zero element is equal to $1$ and is on the $x-th$ place.
For example with $v=5$, $\phi(3) = [0,0,1,0,0]$.\\\\
Now we can see that $k'(x,y) = <\phi(x),\phi(y)> = 1 \Leftrightarrow x=y$.\\\\
As $k'$ is an inner product and also behaves identically as $f$, thus we can write:
\begin{equation}
    k(x,y) = \sum_{i=1}^m\sum_{j=1}^nk'(x,y) = \sum_{i=1}^m\sum_{j=1}^n \phi(x)^T\phi(x)
\end{equation}
Which shows that $k$ is a valid kernel.
