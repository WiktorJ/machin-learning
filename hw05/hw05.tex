\section{Problem 1}
\subsection{}
We know that sum of convex functions is also convex, so in this case it's enough to prove 
that each component is convex.
\begin{itemize}
    \item $x^2$ - Convex because second derivative ($2$) is always non negative
    \item $2y$ - Convex because linear functions are convex.
    \item $cos(sin(\sqrt{\pi}))$ - Convex because constant are convex (it's also just linear function).
    \item $-\min\{-x^2, log(y)\} = \max(x^2, -log(y))$ - Convex because max of convex functions is convex (proven later). 
    $-log(y)$ is convex because second derivative $\frac{1}{x^2}$ is always non negative 
    (and logarithm is defined on $(1,50)$.
\end{itemize}
\textbf{$f(x,y)$ is convex}
\subsection{}
\begin{equation}
    f'(x) = x^{-1} -3x^2
\end{equation}
\begin{equation}
    f''(x) = -x^{-2} -6x
\end{equation}
\\
\begin{equation}
    \begin{gathered}
        f''(x) \geq 0 \Rightarrow \\
        -x^{-2} -6x \geq 0 \Rightarrow \\
        -1 -6x^3 \geq 0 \Rightarrow \\
        -1 \geq 6x^3 \Rightarrow \\
        x \leq \sqrt[3]{\frac{-1}{6}} \Rightarrow \\
        x \leq \sim -0.55
    \end{gathered}
\end{equation}
$(-\infty, -0.55) \cap (1, \infty) =  \emptyset \Rightarrow$ \textbf{$f(x)$ is not convex}
\subsection{}
\begin{equation}
    f(x)= - \min\{log(3x + 1), - x^4 - 3x^2 + 8x - 42\} = 
    \max\{-log(3x+1), x^4 + 3x^2 - 8x + 42\}
\end{equation}
The first part of the $\max$ is convex because $-log$ is convex and non-decreasing and $3x+1$ is convex 
and composition of convex non-decreasing function with convex function is convex.\\
The second part of $\max$ is convex because all elements of the polynomial are convex.\\
Since both functions are convex and max of convex functions is convex \textbf{$f(x)$ is convex}
\subsection{}
\begin{equation}
    \begin{gathered}
        \frac{\partial f}{\partial x} = 3yx^2 - 2yx\\
        \frac{\partial f}{\partial y} = x^3 - x^2 +2y + 1\\
        \frac{\partial f}{\partial x^2} = 6yx - 2y\\
        \frac{\partial f}{\partial x \partial y} = 3x^2 - 2x\\
        \frac{\partial f}{\partial y \partial x} = 3x^2 - 2x\\
        \frac{\partial f}{\partial y^2} = 2\\
        M= 
        \begin{bmatrix}
            6yx - 2y & 3x^2 - 2x \\
            3x^2 - 2x & 2
        \end{bmatrix}
    \end{gathered}
\end{equation}
\begin{equation}
    \begin{gathered}
        \begin{bmatrix}
            a & b
        \end{bmatrix} 
        M
        \begin{bmatrix}
            a \\ b
        \end{bmatrix} = 
        a^2(6yx-2y) + 2ab(3x^2 - 2x) +2b^2
    \end{gathered}

\section{Problem 4}
Assumptions
\begin{itemize}
    \item $f$ is convex
    \item $x, y \in \mathbb{R}^n$
    \item $\nabla f(x) = 0, \nabla f(y) = 0$ are local minima
    \item $ f(x) \geq f(y)$
    \item $\lambda \in (0,1)$
\end{itemize}
From assumptions and convexity definition:
\begin{equation}
    f(\lambda x + (1 - \lambda)y) \leq \lambda f(x) + (1 - \lambda) f(y)
    \leq \lambda f(x) + (1 - \lambda) f(x)
    = f(x)
\end{equation}
To write it more clearly:
\begin{equation}
    f(\lambda x + (1 - \lambda)y) \leq f(x)
\end{equation}
Because of our assumptions regarding $\lambda$ we know that 
$\lambda x + (1 - \lambda)y$ lies somewhere "between" $x$ and $y$. 
But we also said that $f(x)$ is a local minimum, which means it has to be
smaller then all the values in it's neighbourhood. 
So the only way to satisfy equation 13 is to set $x=y$ which means 
that there is only one (global) minimum.
